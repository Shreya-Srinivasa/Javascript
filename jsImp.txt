lines on codes are called js statements
should end with semicolon(;)

js keywords
*break
*if - else
*switch

%modulus
++increment
--decrement
+addition
-subtraction

assignment operators:
x = y ~ x = y
x + = y ~ x = x = x + y
x - = y ~ x = x = x - y
x * = y ~ x = x = x * y
x / = y ~ x = x = x / y
x % = y ~ x = x = x % y

prefix and suffix - x++ and ++x are different
